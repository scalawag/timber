  5. def accepts(entryFacets: EntryFacets): Option[Boolean]

    Tests the condition against the EntryFacets specified.

    Tests the condition against the EntryFacets specified. It should always return the same thing given the same input EntryFacets.

    If the condition returns a definitive answer when passed an empty EntryFacets, it means that this is the answer regardless of the input and the condition may be optimized out of the configuration graph as a constant.

    returns

    Some(true) or Some(false) if the test is decisive and None if it can't be sure given the EntryFacets. None will be treated as 'false' for evaluation purposes.
